# Generation 2 — Interface and Bulk Transport Study

## Question

Was the ferrite-layer failure caused by interface penalties or by poor bulk ferrite transport?

## Comparisons

- Baseline diamond device
- Diamond/ferrite transport layer
- Ferrite plus graphene collector
- Graded interfaces
- Composite layers
- Embedded graphene collectors
- Idealized defect sink

## Outcome

Bulk ferrite transport was the primary bottleneck in the tested model. Embedded graphene collection structures were the only family that clearly improved charge collection.

## Sensitivity Audit

An initial sensitivity ranking exaggerated carrier mobility because electric field was used as a proxy. A normalized coefficient audit showed mobility, interface losses, and trap density are all important.

## Design Lesson

The useful mechanism is shorter carrier travel distance, not ferrite self-healing.
