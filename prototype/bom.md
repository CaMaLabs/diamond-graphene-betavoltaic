# Prototype BOM — Non-Radioactive Surrogate Path

This BOM intentionally excludes radioactive materials.

## Core Items

- Commercial CVD diamond sample or detector-grade diamond substrate
- Contact deposition access or suitable conductive contact method
- Graphene film, graphitic-channel access, or surrogate conductive finger pattern
- Low-noise electrometer or source-measure unit
- Shielded test enclosure
- Temperature sensor
- Microscope inspection access
- Probe station or stable contact fixture

## Optional Professional Facility Items

- Diamond microfabrication access
- Raman spectroscopy
- SEM inspection
- Cleanroom lithography support
- Contact-resistance measurement support

## Excluded

- Radioactive source procurement
- Radioactive material handling equipment
- Any non-licensed source preparation steps
