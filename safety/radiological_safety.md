# Radiological Safety Boundary

This project is for simulation, literature review, and professional research planning.

## Allowed Scope

- Non-radioactive surrogate experiments
- Simulation and modeling
- Literature review
- Device architecture planning
- Professional collaboration planning

## Regulated Scope

Any work involving radioactive beta sources must occur only through properly licensed institutions, trained personnel, approved facilities, and compliant source-handling procedures.

## Not Provided Here

This repository does not provide:

- isotope acquisition instructions,
- radioactive material handling instructions,
- source preparation procedures,
- encapsulation steps for radioactive material,
- instructions for bypassing licensing or regulation.

## Recommended Development Path

1. Validate collector geometry using non-radioactive excitation.
2. Characterize diamond/graphitic interfaces.
3. Publish simulation and surrogate results.
4. Partner with licensed labs for any source-based validation.
